# 🛒 Shopping Cart App

A fully responsive Shopping Cart built with **React + Vite**, styled using **Tailwind CSS**, and powered by **React Context API** for state management. Features include cart item persistence, animated UI, routing, and toast notifications.


## 🚀 Features

- 📦 View product list with images and prices
- 🛍 Add / remove items from cart
- 🔢 Quantity management
- 💾 Cart persistence using `localStorage`
- 🔁 Client-side routing with `react-router-dom`
- 💬 Toast notifications on actions
- ✨ Smooth animations via `framer-motion`
- 📱 Fully responsive design with Tailwind CSS



## 🛠️ Tech Stack

| Technology     | Role                            |
|----------------|---------------------------------|
| React + Vite   | Frontend framework              |
| Tailwind CSS   | Styling                         |
| React Context  | Global state for cart           |
| React Router   | Navigation (Products / Cart)    |
| React Hot Toast | Notifications                  |
| Framer Motion  | Animations / Modal transitions  |
| LocalStorage   | Persist cart across sessions    |



## 🔧 Setup Instructions

```bash
# Clone the repo
git clone https://github.com/your-username/shopping-cart-react.git
cd shopping-cart-react

# Install dependencies
npm install

# Run the app
npm run dev
````

Then visit: [http://localhost:5173](http://localhost:5173)



## 📁 Project Structure

```
src/
├── assets/            # Images / icons
├── components/        # UI components (Navbar, ProductList, Cart, etc.)
├── context/           # React Context for Cart
├── data/              # Static product data
├── App.jsx            # Main app with routing
└── main.jsx           # Entry point
```
